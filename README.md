# QuoteGenerator - author: Prathamesh Charoskar

This app is made using express JS library it is an open source application 

Steps to run this project:-
1.Copy the files
2.On terminal/powershell/cmd/bash run below commands
   a.npm i
   b.npm i express
   c.npm i randomquote-api
   d.npm i cookie-parser
3.On your browser run 'localhost:3000/'
